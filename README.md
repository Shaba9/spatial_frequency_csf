
# Spatial Frequency Sensitivity (Contrast Sensitivity Function)
===========================================================

This experiment measures visual contrast sensitivity across spatial frequencies.

## Requirements:
- Python 3.x
- PsychoPy

---

## Setup

```bash
# 1) Create and activate a virtual environment (recommended)
    py -3.11 -m venv .venv
    .venv\Scripts\activate

# 2) Install dependencies
    pip install psychopy

# 3) Run the experiment:
    python csf_experiment.py
```

**Output:**
- CSV file saved in the /data folder with accuracy per condition
